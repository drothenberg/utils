------------------------
OVAL Checker
------------------------

The OVAL Checker is a Schematron based utility that will flag common mistakes and poor stylistic decisions in OVAL Definition documents. The list of issues that the utility detects and reports will be ever evolving as the community identifies OVAL authoring best practices. The current listing of authoring style guidelines is based off the OVAL Repository Style Guide (http://oval.mitre.org/repository/about/style.html).


-- REPORTING PROBLEMS --

To report a problem with the OVAL Checker, please post a bug report on the OVAL Checker SourceForge site (http://sourceforge.net/projects/ovalutils/).
